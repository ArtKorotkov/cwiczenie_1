Możliwości formatowania tekstu w HTML.

  • Pogrubienie:< b > < / b >
  • Kursywna:< i > < / i >
  • Podkreślenie:< u > < / u >
  • Przekreślelie: < strike > < / strike >
  • Indeksgórny:< sup > < / sup >
  • Indeksdolny:< sub > < / sub >
  • Przejście do nowego wiersza w tekscie: < br >
  • Zapobieganie automatycznemu przenoszeniu fragmentu tekstu do nowego wiersza: < nobr > < / nobr >
  • Spacja niełamliwa: & nbsp

Nagłówki w treści strony WWW.

  1. < H1 > Będzie największym nagłówkiem.
  2. < H2 >...< H5 > Będzie coraz mniejsze.
  3. < H6 > Będzie największym nagłówkiem

Możliwości funkcj font.

  1. < font size ="wielkość czcionki" >
  2. < font color ="kolor czcionki" >
  3. < font face ="rodzaj czcionki" >

3 Najprostsze rodzaje kolorów do użycia w funkcji font.

  1. Nazwą: red-czerwony, green-zielony, blue-niebieski
  2. RGB: rgb(255,0,0)-czerwony, rgb(0,255,0)-zielony, rgb(0,0,255)-niebieski
  3. Hexadecymalny, #RRGGBB: #FF0000-czerwony, #00FF00-zielony, #0000FF-niebieski.
